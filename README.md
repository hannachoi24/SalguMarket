## About The Project

![살구마켓_인트로-001](https://user-images.githubusercontent.com/65268003/144228031-7b4294ee-0876-4f8d-9e2d-185d0207d7b3.png)

살구마켓은 당일 제조, 유통 및 사용되어 신선도에는 이상이 없는 식료품들과 쓰다 남아 버려지는 자재들을 저렴한 가격으로 거래하는 플랫폼입니다.

### Features:
* 회원가입 / 로그인
* 물품 등록
* 완제품 / 식재료 / 폐자재 상품 리스트
* 번개장터
* 물품명 검색
* 물품 상세
* 마이페이지
